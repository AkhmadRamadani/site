Pengajuan Lembur

Melakukan lembur tapi Anda bingung bagaimana cara pengajuannya? Silahkan mengikuti langkah-langkah berikut:

1.

Klik pada ikon menu lembur

2.

Selanjutnya Anda dapat klik tombol (+) di pojok kanan bawah untuk membuat pengajuan lembur

3.

Silahkan memilih pengaturan lembur terlebih dahulu

4.

Isikan informasi mengenai tanggal & waktu memulai dan selesai lembur

5.

Nominal lembur akan terisi secara otomatis tergantung pengaturan perusahaan

(Note : Apabila informasi nominal tidak muncul sewaktu user pengajuan lembur maka terdapat pengaturan perusahaan yang mengaktifkan untuk tidak menampilkan nominal)

6.

Masukkan alasan lembur pada kolom alasan

7.

Isikan lampiran file bukti pendukung berupa foto atau file

8.

Selanjutnya tekan tombol 'Buat Pengajuan'
